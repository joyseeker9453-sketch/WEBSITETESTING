# 晨昕診所官網 — 內容更新說明（給醫師）

網址：`https://診所網址/admin`（加入書籤）
登入：用診所的 GitHub 帳號按一下即可。

## 發休診公告

1. 左側點「**診所公告**」→ 公告列表
2. 按「新增公告」（或點舊公告直接改）
3. 填三格：標題、公告日期（如 2026-07-07）、內容
4. 右上角按「**儲存 / 發布**」→ 約 1 分鐘後官網首頁出現

**下架公告不用刪除**：把該則公告的「顯示在首頁」開關關掉即可，歷史公告都會留著，明年同節日可以改日期重新打開。

## 發健康新知文章

1. 左側點「**健康新知**」→ 右上「新增文章」
2. 填：標題、發布日期、分類（下拉選「慢病體重管理」或「身心科」）、作者、摘要
3. 內文欄像 Word 一樣直接打字；圖片直接拖進來就會自動上傳
4. 按「**發布**」→ 約 1 分鐘後上線，文章自動出現在對應分類，底部自帶 Threads / Facebook / LINE 分享按鈕

## 常見問題

- **按了發布沒馬上看到？** 部署需要約 1 分鐘，重新整理頁面即可。
- **打錯字想改？** 點進該篇文章或公告，改完再按發布，會直接覆蓋。
- **想要新的文章分類？** 這是唯一需要找管理員的事（在設定檔加一行字）。
- **每一次修改都有紀錄**，改壞了都救得回來，放心編輯。
